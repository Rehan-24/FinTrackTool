### 1. Mystery "0" Text Below Categories
**Problem:** Every category showed a "0" below the progress bar, even when there were no upcoming charges.

**Root Cause:** The parent `<div>` had `style={{fontSize: '0'}}` which was a CSS hack to prevent spacing issues. This hack created a text node displaying "0".

**Fix:** Removed all the unnecessary inline styles:
- Removed `fontSize: '0'` from parent div
- Removed `fontSize: '1rem'` overrides from child divs
- Removed `display: 'block'` and `display: 'flex'` inline styles
- Cleaned up the code to use standard CSS without hacks

### 2. Redundant Upcoming Card
**Problem:** The yellow "Upcoming" card was added in v4.4.4 for debugging but wasn't needed long-term.

**Fix:** 
- Removed the yellow "Upcoming" card from dashboard
- Changed grid from 5 columns back to 4 columns
- Kept the upcoming charges info shown under each category
