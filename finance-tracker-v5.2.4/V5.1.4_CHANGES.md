## Setup Instructions

### Step 1: Install Dependencies

```bash
npm install
```

This installs `sharp` for icon generation.

### Step 2: Generate PNG Icons

Run the icon generation script:

```bash
npm run generate-icons
```

This creates:
- `icon-192.png` (192x192)
- `icon-512.png` (512x512)
- `apple-touch-icon.png` (180x180)

### Step 3: Deploy

Deploy to your hosting service (Render, Vercel, etc.)

### Step 4: Test on Mobile

1. Visit your deployed app on your phone
2. Follow "Add to Home Screen" instructions above
3. Launch from home screen - enjoy the app experience!

## Troubleshooting

### Icons Not Generating?

```bash
# Install sharp manually
npm install sharp

# Run generation script
npm run generate-icons
```

### Icon Not Showing on Mobile?

1. Check files exist in `/public`
2. Clear browser cache
3. Re-add to home screen
4. Check console for errors

### App Opens in Browser?

- Make sure `display: "standalone"` in manifest.json
- Re-add to home screen (remove old one first)
- Some browsers have limited PWA support
